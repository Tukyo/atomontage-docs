# Shortcuts

## General

* `Ctrl + Z` Undo (limited to some actions)
* `Ctrl + Y` Redo
* `Ctrl + R` Reset lua state
* `Ctrl + K` See list of quick actions

## Hierarchy

* `F` Focus selected object in hierarchy
* `Esc` Deselect object in hierarchy
* `Delete` Delete selected object
* `Hold Ctrl` Toggle between snapping and non snapping transform movement 
* `Ctrl + D` Duplicate

## Camera

* `Shift` Move faster
* `Q`/`E` Move camera on alternative axis
* `Middle Mouse Button` Focus the camera on the clicked point

## Shape selection

* `V` Paint
* `C` Fill
* `R` Remove
* `Ctrl + C` Copy
* `Ctrl + V` Paste
* `Ctrl + X` Cut
