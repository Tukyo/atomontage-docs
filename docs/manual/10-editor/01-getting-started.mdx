## Login | Create Account
To get started with the Atomontage editor, you'll need to log in or create an account. Follow these steps:

1. Go to the [Atomontage website](https://www.atomontage.com/).
2. Click on the "Make Montage" button in the top right corner.
3. Login or create an account, using your email or Google auth.
4. From here you can create montages, or add assets to your library.

## Create a Montage
Once you are in the [Maker Dashboard](https://makers.atomontage.app/), you can create your own Montage, a project that you have complete control over.
